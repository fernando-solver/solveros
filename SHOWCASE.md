## Why share?

Three reasons:

1. **It helps others** — people considering Solverkitty look here to see "is this for me?"
2. **It helps the project** — we learn what real users actually use, what's missing, what to build next
3. **It helps you** — backlinks, discovery, network effect

We don't gatekeep — students, hobbyists, professionals, anyone using Solverkitty for any real purpose is welcome.
