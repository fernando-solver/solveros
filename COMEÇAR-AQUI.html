Comecar aqui

Um workspace
com memoria,
em 5 minutos.

Voce vai instalar o Solver Filesystem e deixar seu agente reconhecer voce, seus projetos, e o que voce estava fazendo ontem — sem precisar explicar de novo.

Nota editorial · pre-requisitos

Duas coisas precisam estar instaladas antes de comecar.

  1. Python 3.10 ou mais novo. baixar aqui. No Windows, marque a caixinha "Add python.exe to PATH".
  2. Claude Code. baixar aqui.

Ja tem ambos? Siga. Faltou um? Instale e volte.

01
Extrair

Coloque o kit numa pasta

Voce recebeu um arquivo .zip. Extraia em uma pasta onde o workspace vai viver — Desktop ou Documentos servem bem.

Evite Extrair dentro de OneDrive, Google Drive, iCloud ou Dropbox. A sincronizacao corrompe o banco SQLite.

Bom exemplo: C:\Users\SeuNome\Desktop\meu-workspace

02
Abrir

Abra o Claude Code na pasta

Escolha seu sistema:

Clique com o botao direito na pasta do workspace:

  • Windows 11 → "Abrir no Terminal"
  • Windows 10 → "Abrir janela do PowerShell aqui" (Shift + clique direito)

No terminal, digite:

claude

Abra o Terminal (Cmd+Space → "Terminal"). Navegue ate a pasta e abra o Claude Code:

cd ~/Desktop/meu-workspace claude

Abra o terminal dentro da pasta (pelo file manager ou manualmente) e digite:

claude
03
Invocar

Cole a frase magica

No Claude Code que acabou de abrir, copie e cole:

configure este workspace

O agente detecta que o workspace e novo e inicia a configuracao assistida.

04
Responder

Quatro perguntas, dois minutos

O agente vai perguntar:

  • Como voce se chama
  • Que nome quer dar para o agente
  • Qual o papel desse agente
  • Qual sua area de trabalho

No fim, ele pergunta se voce quer criar o primeiro projeto. Diga sim — e o jeito mais rapido de sentir o valor.

›
Dia a dia

Comandos depois da configuracao

  • /nova-pasta — cria um projeto novo com a estrutura padrao
  • /compartilhar X — exporta projeto em ZIP curado (scrub de senhas, HANDOFF.md)
  • /dashboard — retrato visual do workspace (calendario, horas, objetivos)
  • /fechar-dia — consolida atividades do dia no DIARIO.md
  • /status — panorama rapido

Voce nao precisa decorar. Tambem funciona em linguagem natural: "compartilha esse projeto com o Pedro", "me da um panorama".

Problemas comuns

O Claude Code nao abre. Confirme que instalou em claude.com/product/claude-code.

Diz que Python nao foi encontrado. Python nao esta no PATH. No Windows, reinstale marcando "Add python.exe to PATH", feche e abra tudo de novo.

O agente nao entendeu a frase. Digite manualmente: /setup

Extrai errado e os arquivos ficaram soltos. Extraia de novo garantindo que fiquem em UMA pasta (CLAUDE.md, pmo_db.py, README.md todos juntos na raiz).

Outro problema? Acesse fernandosolver.com.br para suporte e documentacao.

Prefere instalar via terminal

Abra o terminal dentro da pasta do kit e rode:

python instalar.py "/caminho/do/seu/workspace"

Ou leia o README.md para arquitetura completa.
