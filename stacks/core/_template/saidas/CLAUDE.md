# saidas/

**Funcao:** entregaveis finais consolidados.

## O que vai aqui

Documentos prontos pra serem usados/enviados/apresentados:
- Relatorios consolidados (HTML, PDF)
- Apresentacoes
- Propostas
- Resumos executivos

## Convencao de nome

`<tipo>_YYYYMMDD.<ext>` — exemplo `relatorio_20260502.html`,
`proposta_20260415.pdf`. Permite ordenacao cronologica natural.

## Versionamento

Quando substituir um entregavel, **mover o antigo pra `_archive/<ano>Q<n>/`**
antes de salvar a nova versao. Nunca apagar sem archive — Camada C do combate
a entropia.
